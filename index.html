Правила:

1. Если число MIN меньше '-999', будет присвоено '-999', если больше '998' - будет присвоено '998'

2. Если число MAX больше '999', будет присвоено '999', если меньше '-998' - будет присвоено '-998'

3. Нечисловое значение будет заменено на 'MIN = 0', 'MAX = 100'

4. Максимальное число должно быть больше минимального
